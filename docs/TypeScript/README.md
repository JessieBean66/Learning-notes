### 变量声明
因为TypeScript是JavaScript的超集，所以它本身就支持let和const。const是对let的一个增强，它能阻止对一个变量再次赋值。

了解ES6新增的这两个变量声明的使用和场景，就能省略在TypeScript中的使用，原理一样。
